macOS

Mac 如此强大好用，macOS 这一操作系统功不可没。它为 Mac 特别打造，采用了软硬件协同设计，能帮助你得心应手地处理各种事情。

iCloud

iCloud 可以安全存储你的照片、视频、文稿、信息、音乐、app 等内容，并能在你的各种设备上同步保持更新。因此，你可以更自由、更方便地访问你想要的内容。

进一步了解 iCloud

照片

能让你日益庞大的照片图库变得井井有条，并易于查看。还能帮助你美化图像，创作精彩照片。更有 iCloud 照片图库，让你可以将值得长久珍藏的照片和视频保存在云端。

进一步了解照片 app

iMovie

以前所未有的方式讲述精彩故事。精简的设计与直观的剪辑功能，让你轻松制作精美的 4K 影片和颇具好莱坞风格的预告片。你还可以通过 iMovie Theater 在各种设备上欣赏这些作品。

进一步了解 iMovie

库乐队

在 Mac 上制作绝妙旋律，简单到离谱。它拥有直观的界面，以及一套完整的声音资源库，让你能够如专业制作人一般学习、演奏、录制和共享乐曲，一切就这么简单轻松。

进一步了解库乐队

Pages

这款功能强大的文字处理软件，能满足你的各种所需，令制作出的文档看起来漂亮，读起来也漂亮。你可在 Mac 和 iOS 设备之间无缝切换，还可与使用 Microsoft Word 的其他人轻松协作。

进一步了解 Pages

Numbers

创建精密的电子表格，添加引人注目的交互式图表、表格和图像，让你的数据生动显现。你可在 Mac 和 iOS 设备之间无缝切换，还可与使用 Microsoft Excel 的其他人轻松协作。

进一步了解 Numbers

Keynote

通过赏心悦目的演示文稿，让你的想法得以生动呈现。运用多种强大工具和炫目效果，让你的观众目不转睛。你可在 Mac 和 iOS 设备之间无缝切换，还可与使用 Microsoft PowerPoint 的其他人轻松协作。

进一步了解 Keynote

Safari 浏览器

Safari 浏览器的各种创新功能让你尽享网络的更多精彩。节能技术让你上网浏览的时间更长，内置隐私功能比以往更为强大，你可以在各种设备之间无缝切换浏览，还可以轻松分享你喜爱的页面。

专业 App

对于想要尽情发挥创意的专业人士而言，这里有众多为业界所推崇的 app，可以用来更好地掌控音乐与影片的剪辑、处理和输出。

  • Final Cut Pro X
  • Logic Pro X
  • Motion
  • Compressor
  • MainStage 3

Final Cut Pro X

为了满足如今剪辑师们四溢的灵感，Final Cut Pro 提供开创性的视频剪辑功能、强大的媒体组织整理能力和杰出的性能，并针对 Mac 电脑和 macOS High Sierra 进行了全面优化。

进一步了解 Final Cut Pro X

Logic Pro X

Logic Pro 在你的 Mac 上构建起一座完备的录音和 MIDI 工作室，以别开生面的方式满足你作曲、录音、编辑和混音的各种所需。有了大量功能齐备的插件，以及数千种声音和循环乐段，无论你想创作哪种音乐，都能为你提供从灵感初现到完成最后的母带的各种所需。

进一步了解 Logic Pro X

Motion

Motion 是一款强大的动态图形工具，可帮助你轻松制作电影风格的 2D 及 3D 字幕、流畅的转场和逼真的特效，一切效果即时可见。

进一步了解 Motion

Compressor

从 Final Cut Pro 中导出项目的功能变得更强大、更灵活。你可以自定义输出设置，使用分布式编码功能提高工作效率，并且可以轻松将影片打包提交至 iTunes Store。Compressor 是你自定义编码的出色搭档。

进一步了解 Compressor

MainStage 3

专为现场演出打造的全屏幕界面，灵活的硬件操控，以及众多与 Logic Pro X 完全兼容的插件和音效，将你的 Mac 变身现场演出装备。

进一步了解 MainStage 3
